Thank you for installing {{ .Chart.Name }} {{ .Chart.Version }}

## What to do next

1. **Configure Secrets**
   You must provide secrets before deployment:

   ```bash
   # Create namespace (if not using --create-namespace)
   kubectl create namespace {{ .Values.namespace.name }}

   # Create secrets
   kubectl create secret generic crms-secrets \
     --namespace {{ .Values.namespace.name }} \
     --from-literal=SPRING_DATASOURCE_PASSWORD='your-db-password' \
     --from-literal=MINIO_ACCESS_KEY='your-minio-user' \
     --from-literal=MINIO_SECRET_KEY='your-minio-password' \
     --from-literal=APP_SECRET_KEY='your-32-char-secret' \
     --from-literal=JWT_SECRET='your-64-char-hex-secret'
   ```

2. **Install the chart**
   ```bash
   helm install crms ./helm \
     --namespace {{ .Values.namespace.name }} \
     --set backend.image.repository={{ .Values.global.imageRegistry }}/{{ .Values.backend.image.repository }} \
     --set frontend.image.repository={{ .Values.global.imageRegistry }}/{{ .Values.frontend.image.repository }}
   ```

3. **Configure DNS**
   Add the following entry to your DNS or /etc/hosts:
   ```
   <INGRESS-IP>  {{ if .Values.ingress.hosts }}{{ index .Values.ingress.hosts 0 "host" }}{{ else }}crms.example.com{{ end }}
   ```

4. **Access the application**
   - Web UI: https://{{ if .Values.ingress.hosts }}{{ index .Values.ingress.hosts 0 "host" }}{{ else }}crms.example.com{{ end }}
   - API: https://{{ if .Values.ingress.hosts }}{{ index .Values.ingress.hosts 0 "host" }}{{ else }}crms.example.com{{ end }}/api/

5. **MinIO Console** (for object storage administration)
   ```bash
   kubectl port-forward -n {{ .Values.namespace.name }} svc/minio-console 9001:9001
   # Access at http://localhost:9001
   ```

## Upgrading

```bash
# Upgrade to a new version
helm upgrade crms ./helm \
  --namespace {{ .Values.namespace.name }} \
  --set backend.image.tag=v1.1.0
```

## Uninstalling

```bash
# Warning: This will delete all data!
helm uninstall crms --namespace {{ .Values.namespace.name }}
kubectl delete pvc -n {{ .Values.namespace.name }} --selector app.kubernetes.io/name=crms-ci-g-uk
```
